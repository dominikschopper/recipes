# Rote Inge

kann eindrucksvoll in einer Glasschüssel oder in kleinen Gläschen serviert werden.

## Zutaten
- 300gr Joghurt
- 200ml Sahne
- 200gr Meringue oder Baisers
- 300-400gr gefrorene Himbeeren
- 1P Vanillezucker

## Zubereitung

1. Sahne steif schlagen
2. Joghurt unter die Sahne rühren
3. Meringue in grobe Stücke zerbrechen (nicht zu klein) und unter die Sahne-Joghurt Mischung heben
4. in eine Schüssel (oder die Gläser) eine Schicht der weißen Masse geben, darauf Himbeeren, dann
  immer abwechselnd so weiter. Sollte ca. 4 Schichten geben (abhängig vom Gefäß)
5. Mind. 30min max 90min kalt stellen.
6. servieren ;)
